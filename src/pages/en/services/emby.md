You can create an API key from inside Emby at `Settings > Advanced > Api Keys`.

Allowed fields: no configurable fields for this widget.

```yaml
widget:
    type: emby
    url: http://emby.host.or.ip
    key: apikeyapikeyapikeyapikeyapikey
```
