You can create an API key from inside Jellyfin at `Settings > Advanced > Api Keys`.

Allowed fields: no configurable fields for this widget.

```yaml
widget:
    type: jellyfin
    url: http://jellyfin.host.or.ip
    key: apikeyapikeyapikeyapikeyapikey
```

*Added in v0.1.0*
